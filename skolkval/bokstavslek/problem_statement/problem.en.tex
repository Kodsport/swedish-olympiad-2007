\problemname{Letter Game}

You are given a string of letters where all letters are distinct and chosen from the uppercase letters \texttt{A}-\texttt{Z}. You are to change the order of the letters by performing at most $n$ swaps so that the resulting string comes as early as possible in alphabetical order.
Each swap means that you exchange two letters that are adjacent.

\section*{Input}
The first line contains a string, the initial letter sequence.

The next line contains the integer $n$ ($1 \leq n \leq 100$), the number of swaps you may perform.

\section*{Output}
Output a single string: the letter sequence that comes first in alphabetical order among those that are reachable.

\section*{Scoring}
Your solution will be tested on a set of test groups.
To earn points for a group, you must pass all test cases in that group.
